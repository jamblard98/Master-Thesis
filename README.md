# Master-Thesis
Master Thesis : Changes in Southern Ocean temperature since 1900: comparison between historical data and a NEMO model simulation

Voici le document qui se rapporte à ce projet : https://thesis.dial.uclouvain.be/entities/masterthesis/c6515347-5315-4cc6-b70e-705179b2f485
* combine_yamazaki_nemo_panels_OK.py : produit les statistiques nécessaires à l'analyse ainsi que la figure 14
* en4_polar_coverage_2deg_OK.py : produit la figure 7
* wod_polar_coverage_2deg_OK.py : produit la figure 6
* extract_nemo_at_yamazaki_positions_OK.py : extrait nemo aux mêmes positions que les observations historiques (voir section 3.2.3)
* extract_wod_en4_observations_OK.py : extrait les observations historiques en éliminant les doublons et en y associant une valeur depuis la climatologie de Yamazaki et al. (2025) (voir section 3.1.1 et 3.2.2)
* map_secteurs.py : produit la figure 8 illustrant le découpage en secteur
* nemo_and_histo_maps_OK.py : produit les cartes d'anomalie pour les données de simulation
* nemo_and_histo_profiles_OK.py : produit les profils d'anomalie les diagrammes/histogrammes pour les données de simulation
* yamazaki_and_histo_maps.py : produit les cartes d'anomalie pour les données observationnelles
* yamazaki_and_histo_profiles_OK.py : produit les profils d'anomalie les diagrammes/histogrammes pour les données observationnelles
